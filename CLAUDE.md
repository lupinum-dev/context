# Claude Code Guide

Use `AGENTS.md` as the canonical development guide for this repository.

Do not rely on old Prompt Tower paths such as `src/api`, `src/providers`, or `src/services`. The current architecture is `src/core`, `src/app`, `src/vscode`, `src/shared`, and `src/webview`. Local Git diff selection is part of the product; GitHub API integrations are not.
